Cansat TechNova - Satelites enlatados 2025

Proyecto para simular una misión espacial, manteniendo vivo un tripulante (huevo) mientras recolecta datos de altura, temperatura, presión y GPS. Desarrollado para el curso-Concurso Mundial de Satelites enlatados 2025 con sistema de autogiro para reducir velocidad de caída.

Tecnologías Utilizadas

ESP32 LoRa RYLR BMP180 MPU6050 Fusion360 EasyEDA

Detalles del Proyecto

Descripción

El Cansat TechNova fue desarrollado para el curso-Concurso Mundial de Satelites enlatados 2025. El objetivo principal era simular una misión espacial donde el satélite debía mantener vivo a un tripulante (representado por un huevo) durante una caída desde aproximadamente 400 metros de altura.

El sistema utiliza módulos LoRa RYLR para comunicación a larga distancia, logrando hasta 600 metros de comunicación continua. Todo el diseño fue realizado con impresión 3D y madera MDF, integrando sensores BMP180 y MPU6050 para recolección de datos ambientales.

Características

  • • Sistema de supervivencia para tripulante (huevo)
  • • Recolección de altura, temperatura y presión
  • • GPS integrado para localización
  • • Sistema de autogiro para reducir velocidad de caída
  • • Transmisión de datos a estación terrena
  • • Comunicación LoRa hasta 600 metros
