Agrosync: Hackathon Talento-Tech 2025

Plataforma web desarrollada para ayudar a los agricultores a organizar sus cultivos e implementa LLMs para asesoramiento personalizado. Creada durante el Hackathon Talento-Tech 2025 como una solución integral para la gestión agrícola moderna.

Tecnologías Utilizadas

Django Python HTML CSS Gemini API MySQL

Detalles del Proyecto

Descripción

Agrosync fue desarrollado durante el Hackathon Talento-Tech 2025 como una solución integral para el sector agrícola. El producto busca ayudar al agricultor a poder organizar sus cultivos de manera eficiente y moderna.

La plataforma implementa LLMs (Large Language Models) para que puedan asesorar a los usuarios, proporcionando recomendaciones personalizadas. El sistema fue desarrollado como una página web completa con todas las herramientas necesarias para que el agricultor pudiera hacer uso de ellas.

Características

  • • Organización inteligente de cultivos
  • • Asesoramiento con inteligencia artificial
  • • Integración con Gemini API
  • • Plataforma web completa fullstack
  • • Base de datos MySQL para gestión de datos
  • • Desarrollo con Django Framework
